# Chromosome Visualization and Annotation Plotting
This repository contains a Python script that visualizes chromosomes and their corresponding gene annotations based on scores. The script generates a graphical representation of chromosomes, highlights specific regions, and assigns colors to gene annotations according to predefined score ranges.

## Prerequisites
Make sure you have the following libraries installed in your Python environment:
- `matplotlib`
- `pandas`
- `numpy`

You can install them via `pip`:
```sh
pip install matplotlib pandas numpy
